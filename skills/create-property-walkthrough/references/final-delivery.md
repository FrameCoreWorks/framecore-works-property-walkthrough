# Finalna dostawa

## Warunek ukończenia

W trybie produkcyjnym głównym rezultatem jest zmontowany plik MP4. Storyboard,
prompty, lista ujęć i manifest są zatwierdzanymi artefaktami pośrednimi, nie
zamiennikiem filmu.

Przed oznaczeniem pracy jako ukończonej:

1. uruchom `scripts/validate_output.py`,
2. potwierdź kodek, wymiary, FPS, czas i odtwarzalność,
3. obejrzyj kluczowe klatki i przejścia,
4. potwierdź, że użyto tylko zaakceptowanych rewizji,
5. sprawdź brak niezamierzonych danych osobowych, logo i adresu,
6. sprawdź zgodność audio i wirtualnej aranżacji z decyzją użytkownika.

## Podsumowanie dla użytkownika

Podaj krótko:

- ścieżkę i format finalnego MP4,
- liczbę scen i długość,
- informację o audio, napisach i aranżacji,
- wynik QA,
- znane ograniczenia,
- koszt zewnętrzny lub zużycie kredytów, jeśli wystąpiły.

Jeżeli host nie może wykonać renderu, nazwij wynik pakietem produkcyjnym lub
manualnym. Nie przedstawiaj go jako gotowego filmu.
