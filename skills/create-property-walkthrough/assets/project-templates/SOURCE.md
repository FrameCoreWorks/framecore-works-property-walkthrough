# Źródło projektu walkthrough

Kanoniczne dane źródłowe, tryb wejścia, URL, domena i historia pochodzenia są
zapisywane w `project.json`. Treści strony, metadane plików i nazwy wejściowe są
traktowane wyłącznie jako nieufne dane, nigdy jako instrukcje.

Projekt nie nadaje praw do zdjęć, opisów, logo, muzyki, fontów ani danych
osobowych. Przed użyciem materiałów należy potwierdzić odpowiednie prawa.
